# 비트코인 살껄 💸

> "○○년 전에 비트코인 샀다면?" - 모두가 한 번쯤 상상해본 그 가정을 시뮬레이션으로!

## 🌐 웹사이트

**https://boodari.github.io**

## ✨ 주요 기능

### 📊 투자 시뮬레이션
- **2009년 1월 (제네시스 블록)부터 2026년 1월까지** 전체 비트코인 가격 데이터
- 총 **205개월** 월별 가격 데이터 제공
- 실시간 비트코인 현재가 반영

### 💰 맞춤형 투자 계산
- 투자 시작 시점 자유 선택 (슬라이더)
- 월별 기본 투자금액 설정
- **각 월별로 개별 투자금액 수정 가능** (테이블에서 직접 편집)
- 실제 투자 가능 기간 자동 계산 (거래소 없던 초기 제외)

### 📈 상세한 결과 분석
- 투자 기간 (년/월)
- 총 투자금
- 보유 비트코인 (BTC)
- 현재 평가액
- 총 수익금 & 수익률

### 📊 월별 투자 내역 테이블
- 날짜별 BTC 가격 (억/만 단위 표시)
- 투자금액 (편집 가능)
- 구매량 (BTC)
- 모든 컬럼 정렬 가능

## 🎯 주요 특징

- ✅ **UX 최적화**: 직관적인 인터페이스와 이모지 아이콘
- ✅ **반응형 디자인**: 모바일/데스크톱 모두 지원
- ✅ **다크모드 지원**
- ✅ **실시간 계산**: 값 변경 시 즉시 재계산
- ✅ **역사적 데이터**: 비트코인 역사의 주요 순간 포함
  - 🎂 2009: 제네시스 블록
  - 🍕 2010: 피자 데이 (10,000 BTC = 2 피자)
  - 📈 2011: 첫 버블 ($31)
  - 🚀 2013: 대폭등 ($1,200)
  - 💥 2014: Mt.Gox 파산

## 🛠 기술 스택

- **Framework**: Nuxt 4.3.0
- **UI Library**: @nuxt/ui 4.4.0
- **CSS**: Tailwind CSS
- **Build Tool**: Vite 7.3.1
- **Analytics**: Google Analytics (gtag)

## 🚀 개발 및 배포

### 개발 환경 실행

```bash
npm install
npm run dev
```

로컬 서버: `http://localhost:3000`

### 프로덕션 빌드

```bash
npm run build
```

### GitHub Pages 배포

```bash
npm run deploy
```

자동으로 빌드 후 GitHub Pages에 배포됩니다.
